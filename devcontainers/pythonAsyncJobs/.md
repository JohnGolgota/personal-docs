# Python AsyncJobs Example in DevContainer

## requirements

Note: auto-install python requirements

```bash
pip3 install --user -r requirements.txt
```

## run

en diferentes terminales:

```bash
rq worker --url redis://:@cache:6379 &
```

```bash
uvicorn main:app --reload &
```
